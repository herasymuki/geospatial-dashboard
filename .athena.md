## Data Sources (Live / Public APIs)
- ACLED (Armed Conflict Location & Event Data) — REST API (key optional, fallback data included)
- UCDP (Uppsala Conflict Data Program) — REST API (public, no key needed)
- GDELT Project — CSV feed (public, no key needed)
- ReliefWeb (OCHA) — REST API (public, no key needed)
- OpenStreetMap Nominatim (geocoding)

## Dashboard Views
1. **Globe View** — globe.gl with NASA texture, conflict event points, arcs, pulse rings
2. **Deck.gl Map View** — HeatmapLayer + ScatterplotLayer + ArcLayer + HexagonLayer on Leaflet base
3. **Timeline Panel** — chart.js time-series of events per conflict (bar/line toggle)
4. **Drill-Down Panel** — click country/event → detail cards, source breakdown, AI brief button
5. **Filter Sidebar** — date range, conflict type, severity, data source toggle, min fatalities slider
