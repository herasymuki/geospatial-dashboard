## Visualization Libraries
- globe.gl (WebGL globe with NASA Blue Marble texture)
- deck.gl (geospatial layers: HeatmapLayer, ScatterplotLayer, ArcLayer, GeoJsonLayer)
- chart.js (timelines, bar charts, sparklines)
- leaflet.js (2D map fallback / detail drill-down)

## Data Sources (Live / Public APIs)
- ACLED (Armed Conflict Location & Event Data) — REST API
- UCDP (Uppsala Conflict Data Program) — REST API
- GDELT Project — BigQuery public dataset / CSV feeds
- ReliefWeb (OCHA) — REST API (crises, disasters)
- Global Terrorism Database mirror / GTD API
- OpenStreetMap Nominatim (geocoding)
- NASA EONET (natural events that intersect conflict zones)

## Dashboard Views
1. **Globe View** — globe.gl with NASA texture, conflict event arcs, heatmap overlay
2. **Deck.gl Map View** — HeatmapLayer + ScatterplotLayer + ArcLayer + GeoJsonLayer
3. **Timeline Panel** — chart.js time-series of events per conflict
4. **Drill-Down Panel** — click country/event → detail cards, source breakdown
5. **Filter Sidebar** — date range, conflict type, severity, data source toggle
