# Athena App Config — global-conflicts-athena-service

## Identity
- **App Prefix**: global-conflicts
- **GCP Project**: s4d-athena-hosting
- **GCP Region**: us-central1
- **Cloud Run Service**: global-conflicts-athena-service
- **Workspace**: ./gcp_app_builder_workspace

## Stack
- **Framework**: Vue 3 + Vite
- **State**: Pinia
- **Routing**: Vue Router 4
- **3D Globe**: globe.gl (NASA Blue Marble + topology bump + night sky)
- **Geospatial**: Deck.gl 9 (ScatterplotLayer, HexagonLayer, GridLayer, ArcLayer, HeatmapLayer)
- **Charts**: Chart.js 4 + vue-chartjs (Line, Doughnut)
- **Styling**: Pure CSS (dark military intelligence theme)
- **Deployment**: Docker → Cloud Run (nginx:alpine serving Vite build)

## Data Sources (all public, no API key required)
1. **ACLED** — Armed Conflict Location & Event Data (acleddata.com)
2. **UCDP** — Uppsala Conflict Data Program GED API (ucdpapi.pcr.uu.se)
3. **GDELT** — Global Database of Events, Language, and Tone (gdeltproject.org)
4. **ReliefWeb** — UN OCHA humanitarian crises API (reliefweb.int)
5. **RSS Feeds** — Reuters, Al Jazeera, BBC World (via rss2json proxy)

## Views
- 🌍 3D Globe (globe.gl): points, arcs, labels, NASA texture
- 🗺 Deck.gl Map: Scatter / Hexagon / Arc / Grid layers (switchable)
- 🔥 Heatmap: deck.gl HeatmapLayer with adjustable intensity/radius/threshold

## Components
- FilterPanel, ConflictList, NewsPanel (left sidebar)
- GlobeView, DeckMapView, HeatmapView (center — tabbed)
- StatsPanel, TimelineChart, EventTypeChart, RegionBreakdown (right sidebar)
- ConflictDetail (slide-in drawer on event click)
